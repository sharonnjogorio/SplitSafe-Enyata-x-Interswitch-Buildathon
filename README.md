# SplitSafe

A Collaborative Finance (Co-Fi) platform that simplifies how groups manage and settle shared financial obligations.

Built for the **Enyata x Interswitch Buildathon** by Team Stack Innovators.

## What it does
- Track shared expenses across groups
- Calculate individual balances in real time
- Smart Settlement Engine that minimizes the number of transactions needed to settle debts
- Integrated payment flow via Interswitch Web Checkout

## Tech Stack
**Frontend:** React, CSS Modules  
**Backend:** Node.js (Express)
